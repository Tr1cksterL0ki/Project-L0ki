# Legacy content

This repository previously contained a very large “kitchen sink” tuning guide.

It was removed in this updated version because it included high-risk / low-evidence tactics that can:
- reduce security (e.g., disabling Defender, mitigations)
- break system features or updates
- conflict with anti-cheat (VALORANT/Vanguard)
- add significant maintenance burden
